Predictive Policing: Anticipating Criminal Activity

Holmes harnessed machine learning algorithms to predict criminal activities based on historical data:

import pandas as pd
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

data = pd.read_csv("historical_crime_data.csv")
X = data.drop(columns=["crime_type"])
y = data["crime_type"]

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)

clf = RandomForestClassifier()
clf.fit(X_train, y_train)

predictions = clf.predict(X_test)
accuracy = accuracy_score(y_test, predictions)
print(f"Accuracy: {accuracy:.2%}")

Anomaly Detection Algorithm

In the mystery, Sherlock Holmes utilizes an anomaly detection algorithm to find unusual patterns in CCTV footage. The code provided illustrates a simple implementation of this algorithm using OpenCV, a popular library for computer vision tasks:

import cv2
import numpy as np

def detect_anomalies(image, reference_image):
    difference = cv2.absdiff(image, reference_image)
    _, threshold = cv2.threshold(difference, 100, 255, cv2.THRESH_BINARY)
    return threshold

image = cv2.imread("cctv_image.jpg")
reference_image = cv2.imread("reference_image.jpg")
threshold_image = detect_anomalies(image, reference_image)
cv2.imshow("Anomalies", threshold_image)
cv2.waitKey(0)
cv2.destroyAllWindows()
  • The detect_anomalies() function takes two input images: the main CCTV image and a reference image of the same scene without any anomalies.
  • The function computes the absolute difference between the pixel values of the two images using cv2.absdiff(). This results in an image with pronounced differences where anomalies have occurred.
  • Next, the cv2.threshold() function is used to convert the difference image into a binary format. All pixels with a difference of at least 100 units are set to a white value (255), while the others are set to black (0).
  • The function then returns the resulting threshold image highlighting the detected anomalies.
  • When executing the process, the input and reference images are read using cv2.imread(), the anomaly detection is performed, and the contrast image is displayed with cv2.imshow().

Predicting Criminal Activity with Machine Learning

In the LessWrong blog post, a machine learning algorithm is employed to aid in predicting criminal activity based on historical data. The code presented uses the RandomForestClassifier from the Scikit-learn library:

import pandas as pd
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

data = pd.read_csv("historical_crime_data.csv")
X = data.drop(columns=["crime_type"])
y = data["crime_type"]

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)

clf = RandomForestClassifier()
clf.fit(X_train, y_train)

predictions = clf.predict(X_test)
accuracy = accuracy_score(y_test, predictions)
print(f"Accuracy: {accuracy:.2%}")
  • The pandas library is used to read and process the historical crime data from a CSV file.
  • The data is divided into the input features (X) and target labels (y, representing crime type).
  • The dataset is split into a training set and a testing set using train_test_split(). The training set is utilized for building the model, and the testing set allows evaluation of its performance.
  • A RandomForestClassifier is created, trained with the training data (clf.fit()), and then used to predict the crime type of the test data (clf.predict()).
  • The model's accuracy is computed using the accuracy_score() function from Scikit-learn's metrics module, and the result is printed as a percentage.
